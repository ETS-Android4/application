# Welcome to GT-Time, Georgia Tech schedule builder repo

This repository contains the source code for:

* [GT Time](https://play.google.com/store/apps/details?id=com.gttime.android&hl=en_US&gl=US)

Related repositories include:

* [TimetableView](https://github.com/tlaabs/TimetableView/tree/7d58cdd44dd9c562c7e4c67ace022be89d6acdee)

## Prerequisites
* Java (17.0.1+)
* Android Studio IDE (or any builder to generate the project with build.graddle)

## Installation (Developer)
Clone repo\
`git clone https://github.com/cshim31/application.git`

Initiate submodules\
`git submodule update --init`

This command does both cloning and initiating submodule\
`git clone --recurse-submodules https://github.com/cshim31/application.git`


## Installation [Recommended]

Install the [GT Time from the playstore market][store-install].
